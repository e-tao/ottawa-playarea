## Introduction
A web app shows all the palyareas in city of ottawa.  Use can filter their selctions with different craterias.

## Tech stack
 - Frontend 
   HTML, CSS, Javascript, Webpack
 - Backend
   Express, mariadb database
 - 3rd Party API
   MapBox, OpenWeather, Ottawa.ca
