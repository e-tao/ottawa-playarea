## Introduction
A web app shows all the palyareas in city of Ottawa.  Users can filter their selctions with different craterias.

## Tech stack
 - Frontend:
   HTML, CSS, Javascript, Webpack
 - Backend:
   Express, mariadb database
 - 3rd Party APIs:
   MapBox, OpenWeather, Ottawa.ca
